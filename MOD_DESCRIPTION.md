# Smoking Warning Mod — 模组发布简介

## 简介

Smoking Warning Mod（抽烟警示模组）是一个面向 **Minecraft 1.21.1 + NeoForge** 的生存扩展模组。

本模组围绕烟草种植、烟草加工、香烟制作、吸烟行为、成瘾惩罚、肺癌阶段、治疗系统、HUD 展示、村庄禁烟区和烟草村民等内容展开。同时提供 Create 机械动力联动、JEI/EMI/REI/Jade 等可选第三方兼容。

模组的核心设计理念是：**短期收益，长期代价。** 抽烟前期可能带来短暂的正面效果，但随着吸烟次数增加，玩家会进入轻度、中度、重度成瘾，后期会出现咳嗽、肺癌、持续扣血等严重惩罚。

## 主要玩法

### 烟草种植与加工

- 种植烟草种子，收获烟草叶
- 通过晾晒架、研磨台、烟草工作台完成手工加工
- 安装 Create 后可解锁机械化烟草加工流水线

### 香烟系统

- 11 种香烟变体，每种拥有不同效果
- 每支香烟有耐久值，需要持续吸完
- 抽完后获得短暂正面效果，但随着成瘾加深，效果逐渐衰减
- 肺癌阶段继续抽烟会直接扣血

### 成瘾与惩罚

- 轻度成瘾（5-9 支）：轻微咳嗽
- 中度成瘾（10-19 支）：咳嗽加重、最大生命值下降
- 重度成瘾/肺癌（20 支+）：严重咳嗽、继续抽烟扣血

### 治疗系统

- 使用肺部检查仪确诊
- 通过化疗、放疗、靶向治疗逐步推进
- 使用康复计划书进入康复阶段
- 持续戒烟一段时间后肺癌进入缓解状态
- 治疗不会完全清空成瘾记录

### 村庄禁烟区

- 村庄是禁烟区，吸烟会激怒铁傀儡
- 烟草村民可生成并交易烟草相关物品
- 大师级烟草村民出售肺癌治疗手册

## Create / 机械动力兼容

安装 Create 6.0.10+ 后，可获得完整的机械化烟草加工流水线：

- 风扇烘干、石磨研磨、粉碎轮加工
- 机械压制、机械手部署
- 序列组装制作高级香烟
- Create 是**可选兼容**，不是必装前置

## JEI / EMI / REI / Jade 兼容

- **JEI** — 查看全部自定义配方
- **REI** — 查看自定义配方（需 REI 16.x）
- **Jade** — 查看晾晒架进度、方块信息
- 以上均为**可选兼容**，不是必装前置

## 免责声明

本模组**不鼓励**现实中吸烟。吸烟有害健康。

模组中的肺癌和治疗内容均为游戏机制和健康警示创作，**不构成现实医学建议**。如果现实生活中存在健康问题，请寻求专业医生帮助。
